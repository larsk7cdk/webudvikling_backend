# Opgave 2.4

#### Opgave beskrivelse
> Lav et program som viser "Den lille tabel" (altså tallene fra 1 til 10 ganget med hinanden)
>
> Har du brug for at få genopfrisket din folkeskole-matematik, kan du se den lille tabel her: https://www.duda.dk/opgaver/matematik/multiplikation/den-lille-tabel/den-lille-tabel.html

## Solution
Løsningen benytter .Net Core 2.2

#### Run
For at starte programmet kompileres koden. Åbn mappen Debug eller Release og i en kommando prompt skrives
```
dotnet Opgave2_4.console.dll
``` 

#### Test
For at køre unit test kan det gøres fra Visual Studio eller stå i roden af projektet 
```
dotnet test
```
