# Opgave 2.3

#### Opgave beskrivelse
> Lav et program, som slår med en terning. Programmet skal udskrive det tal, terningen viser og blive ved med at slå, indtil det slåer en sekser.
>
> Hvis du er i tvivl om, hvordan du laver et tilfældigt tal i C#, er der hjælp at hente her: http://bfy.tw/GGfg

## Solution
Løsningen benytter .Net Core 2.2

#### Run
For at starte programmet kompileres koden. Åbn mappen Debug eller Release og i en kommando prompt skrives
```
dotnet Opgave2_3.console.dll
```
