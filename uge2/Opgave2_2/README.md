# Opgave 2.2

#### Opgave beskrivelse
> Lav et program, som omregner mellem Celcius og Fahrenheit.
>  
> Du kan se formlen for omregningen her: http://www.metric-conversions.org/da/temperatur/fahrenheit-til-celsius.htm

## Solution
Løsningen benytter .Net Core 2.2

#### Run
For at starte programmet kompileres koden. Åbn mappen Debug eller Release og i en kommando prompt skrives
```
dotnet Opgave2_2.console.dll
``` 

#### Test
For at køre unit test kan det gøres fra Visual Studio eller stå i roden af projektet 
```
dotnet test
```
