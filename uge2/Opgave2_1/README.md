# Opgave 2.1

#### Opgave beskrivelse
> Lav et program som omregner mellem danske kroner og Euro. Antag at kursen altid er 7.45

## Solution
Løsningen benytter .Net Core 2.2

#### Run
For at starte programmet kompileres koden. Åbn mappen Debug eller Release og i en kommando prompt skrives
```
dotnet Opgave2_1.console.dll
``` 

#### Test
For at køre unit test kan det gøres fra Visual Studio eller stå i roden af projektet 
```
dotnet test
```
